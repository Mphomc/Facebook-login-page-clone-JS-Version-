# Facebook Login Page Clone

## Overview

This project is a simple clone of the Facebook login page built using HTML, CSS, and JavaScript.
The goal of this project was to practice front-end development skills and understand how real login interfaces are structured.

## Features

* Facebook-style login layout
* Email/phone and password input fields
* Login button
* Basic JavaScript login simulation
* Responsive layout similar to the desktop version

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose of the Project

I built this project to practice:

* Structuring web pages with HTML
* Styling layouts with CSS
* Handling simple login logic with JavaScript
* Recreating real-world UI designs

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## Disclaimer

This project is for educational purposes only.
It is a UI clone and is not connected to Facebook or any real authentication system.
